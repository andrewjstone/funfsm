### Usage

Add the following to your `Cargo.toml`

```toml
[dependencies]
funfsm = "0.1"
```

Add this to your crate root

```rust
extern crate funfsm;
```

# Description
`funfsm` is a finite state machine library where every state in the FSM is a function.
